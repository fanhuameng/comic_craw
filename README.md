# 漫画抓取
从网站中获取漫画并进行打包，支持cbz格式文件，导入Komga中

# 使用方法
1. 导入python依赖;
2. 去官网找到漫画号,并写入 craw.py comic_list中
3. 执行 python craw.py;
4. 生成漫画在 ./comic 目录下;
5. 手动复制到Komga中;

# 支持 
    漫画屋
比较懒只支持1种；
